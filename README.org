** Specification

*** Entree

Chaque ligne de l'entree represente un sommet, avec, par exemple:

#+begin_src
  A B C
  B C
  C A
#+end_src

qui represente un graphe $G = (V, E)$ tq $G = \{A, B, C\}$ et $V = \{(A, B), (A, C), (B, C), (C, A)\}$.
